== Description ==
Limite les tentatives de connexions à l'administraton de WordPress

== Installation ==
1. Téléchargez le plugin.
2. Faites ajouter une extension
3. téléverser votre extension et installez-la
3. Activez le plugin depuis le menu Extensions dans WordPress.
